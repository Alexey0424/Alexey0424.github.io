# alexey0424.github.io

Personal portfolio of Alexey Velásquez, AI engineer.

Hand-built static site: no framework, no build step, no template. One HTML file,
one stylesheet, one script. The hero background is a custom canvas "signal field"
that reacts to the cursor; interactions respect `prefers-reduced-motion`.

## Run locally

```bash
python -m http.server 8080
# open http://localhost:8080
```

## Stack

- HTML + CSS (OKLCH color, fluid type via clamp) + vanilla JS
- Fonts: Bricolage Grotesque, Martian Mono (Google Fonts)
- Hosted on GitHub Pages
